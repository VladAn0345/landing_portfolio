# Лэндинг проект на Next.js

Простой лэндинг с секциями, созданный на Next.js 14 с TypeScript и Tailwind CSS.

## Структура проекта

```
project/
├── app/
│   ├── layout.tsx          # Корневой layout
│   ├── page.tsx           # Главная страница
│   ├── globals.css        # Глобальные стили
│   └── sections/          # Компоненты секций
│       ├── hero.tsx      # Главная секция
│       ├── about.tsx     # Секция "Обо мне"
│       ├── projects.tsx  # Секция "Проекты"
│       └── resume.tsx    # Секция "Резюме"
├── package.json          # Зависимости проекта
└── README.md            # Этот файл
```

## Секции лэндинга

1. **Главная (Hero)** - Основная информация и приветствие
2. **Обо мне (About)** - Личная информация и опыт
3. **Проекты (Projects)** - Портфолио проектов
4. **Резюме (Resume)** - Опыт работы и образование

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

Проект будет доступен по адресу: http://localhost:3000

## Технологии

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18
